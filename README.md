# mtg-scanner
Lego scanner for mtg cards

## Goal
Create a MTG card dispenser in Lego, connected with a camera to sort and/or digitalize a collection.

## Built with

### Software
* python3
* opencv
* phash python bindings
* brickpi python bindings

### Hardware
* Lego mindstorms
* Raspberry Pi
* Picamera
* BrickPi3
